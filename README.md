# Calculateur de Moyenne

## 📝 À propos du projet

Ce projet est une application web pratique et intuitive permettant de calculer facilement votre moyenne. Ajoutez vos notes, définissez vos coefficients, et obtenez votre résultat instantanément.

## ✨ Fonctionnalités

- Ajout rapide de notes avec ou sans coefficients.
- Calcul automatique de la moyenne générale.
- Interface simple, claire et responsive.
